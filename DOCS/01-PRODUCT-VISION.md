# Продуктовое видение — Danang Expat Hub

> **Last Verified:** 2026-02-17

---

## Миссия

**Убрать языковой и информационный барьер между вьетнамцами и экспатами**, создав AI-driven платформу, где каждый может найти жильё, услуги, товары и людей — общаясь на своём родном языке.

## Целевая аудитория

### Первичная (Primary)
| Сегмент | Описание | Размер |
|---------|----------|--------|
| Экспаты-новички | Приехали в Дананг 0-3 месяца назад. Нужно всё: жильё, байк, SIM, места. | ~2000/мес |
| Экспаты-резиденты | Живут 3-12+ месяцев. Знают основы, ищут комьюнити и сервисы. | ~5000 |
| Digital nomads | Работают удалённо, ищут коворкинги, кафе, нетворкинг. | ~3000 |

### Вторичная (Secondary)
| Сегмент | Описание | Ценность |
|---------|----------|----------|
| Вьетнамские лендлорды | Сдают жильё экспатам. Языковой барьер — главная проблема. | Платный листинг |
| Вьетнамские бизнесы | Кафе, рестораны, услуги для экспатов. | Платное размещение |
| Агенты по аренде | Помогают с жильём и байками. | Комиссия/подписка |

## Ключевые принципы продукта

### 1. AI-First
Каждое действие может быть выполнено через AI ассистента. Пользователь может не знать интерфейс — достаточно написать "найди мне квартиру в Son Tra до $400".

### 2. Zero Language Barrier
- Весь контент переводится AI в реальном времени
- Чат между юзерами автоматически переводит сообщения
- Голосовые → текст → перевод → текст на целевом языке
- Переключатель "показать оригинал" на каждом сообщении

### 3. Community-Driven Content
Контент создаётся пользователями, модерируется AI + ручная модерация. Экспаты добавляют места, лендлорды — объявления.

### 4. Telegram-Native
Полная интеграция с Telegram: авторизация, геолокация, контакты, уведомления, inline-режим (будущее).

## Монетизация (будущее)

1. **Featured Listings** — платное продвижение объявлений (аренда, маркет)
2. **Business Profiles** — платные страницы бизнесов (кафе, коворкинги)
3. **Agent Subscriptions** — подписка для агентов по аренде
4. **Premium AI** — расширенный AI (голосовой ассистент, персональные рекомендации)
5. **Commission** — комиссия с транзакций (аренда через платформу)
